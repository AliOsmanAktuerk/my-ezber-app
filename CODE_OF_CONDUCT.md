# Code of Conduct

My Cami classroom soll ein freundliches, respektvolles Open-Source-Projekt bleiben.

## Erwartetes Verhalten

- respektvoll kommunizieren
- konstruktiv kritisieren
- unterschiedliche Erfahrungsstände ernst nehmen
- Fehler offen und sachlich behandeln

## Nicht akzeptiert

- beleidigende oder ausgrenzende Sprache
- persönliche Angriffe
- absichtliches Stören von Diskussionen oder Entwicklung
- Veröffentlichung privater Daten ohne Zustimmung

Wenn ein Problem entsteht, sollte es ruhig, direkt und mit Blick auf eine faire Lösung angesprochen werden.
